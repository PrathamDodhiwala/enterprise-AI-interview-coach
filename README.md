# 🎯 Enterprise AI Interview Coach

AI-powered interview preparation platform using Machine Learning and Analytics.

---

## Features

### Mock Interview Center

- Role-based questions
- Interview preparation
- Practice sessions

### AI Evaluation Engine

- Interview score prediction
- Communication analysis
- Technical assessment
- Problem solving evaluation

### Analytics Dashboard

- KPI Monitoring
- Score Distribution
- Skill Analysis

### Reports

- Downloadable reports
- Interview insights
- Candidate analytics

---

## Tech Stack

- Python
- Streamlit
- Scikit-Learn
- Pandas
- NumPy
- Plotly
- TextBlob

---

## Installation

```bash
pip install -r requirements.txt
```

Train Model

```bash
python train_model.py
```

Run App

```bash
streamlit run app.py
```
